## 六、公式链

```text
计算载荷质量 = 峰值功率 / 计算载荷比功率
散热系统质量 = 散热板面积 * 面密度
平台质量 = (计算载荷 + 散热 + 电源系统质量) * 平台系数
集成测试/冗余质量 = 前述小计 * AIT/冗余质量系数

其中电源系统质量/成本来自 Task 6：
  - 光伏阵列面积、名义功率、质量
  - 电池名义储能、质量
  - 功率电子额定功率、质量
```

## 七、审计结论

- 吸收 Task 6 后，本地独立拆解得到的单星总质量区间约为 `5.40-11.81 t`；基准情景约 `7.57 t`。
- 该区间仍不是公开规格，而是基于公开功率/散热锚点、工程假设与 Task 6 电源物理链的子系统求和结果。
- 相比旧版 Task 5，最大的结构变化是：电源系统从单一 `$/W` 占位，切换为 `光伏阵列 + 电池 + 功率电子` 三分项，并显式暴露 DoD 对质量的高度敏感性。
- 基准情景下，电源系统本身约为 `1.94 t / $47.75M`，这意味着 Task 5 原来的 `$30M` 电源占位偏低，且未体现电池 DoD 约束。
- 当前总成本的主导项仍是 `光伏阵列` 与 `集成测试/冗余`；质量的主要摆幅来源仍是 `计算载荷比功率`、`电池 DoD` 与 `平台系数`。
