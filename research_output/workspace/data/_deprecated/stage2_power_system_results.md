## 四、公式链

```text
轨道周期 = 2*pi*sqrt((R_earth + h)^3 / mu)
食时占比(beta=0近似) = asin(R_earth / (R_earth + h)) / pi
食区负载能量 = 持续功率 * 食时
电池名义容量 = 食区负载能量 / (DoD * 放电效率)
回充能量 = 食区负载能量 / (充电效率 * 放电效率)
日照段额外回充功率 = 回充能量 / 日照时长
PCDU输出需求 = 持续功率 + 日照段额外回充功率
EOL阵列需求功率 = PCDU输出需求 / 母线效率
阵列面积 = EOL阵列需求功率 / (太阳常数 * EOL电池效率 * 封装/姿态修正)
阵列BOL额定功率 = 阵列面积 * (太阳常数 * BOL电池效率 * 封装/姿态修正)
阵列质量 = 阵列面积 * 面密度
PCDU质量 = 额定功率 / 比功率
```

## 八、审计性判断

- **外部资料支撑较强**：GaAs `30%-32% class`、`radiation-hardened`；LEO 长寿命 Li-ion 在低 DoD 下的循环寿命约束；AI1 `120/150 kW` 与 `~600 km` 的二手一致转述链。
- **工程估算**：阵列面密度、PCDU 比功率与单价、GaAs 阵列 `USD/W_BOL` 缩窄区间。这些值可用于本地核算，但不能伪装成 AI1 官方采购参数。
- **情景参数**：潜在硅基路线的 `20% / 14%` BOL/EOL 口径；其作用只是说明“如果未来采用薄硅 HJT，则面积会显著放大”，不代表当前已采用。
- **待验证缺口**：AI1/Starlink 太阳能化学体系、AI1 阵列收拢面积、真实 EOL 衰减曲线、真实 PCDU 报价、空间级硅路线量产与寿命。

## 九、“星链硅基太阳能电池”专项审查

- 当前公开可核对到的较强材料，更接近“硅基路线在空间应用重新受到重视”与“部分行业分析声称 Starlink 使用硅基晶圆”这两个层次。
- CEA 2025 证明了薄硅 HJT 在空间环境下具备探索价值，但这只说明“硅基路线并非物理上不可能”，不说明 SpaceX 已采用该路线。
- 本轮检索未找到 SpaceX 官网、Starlink 官方资料、FCC 技术披露或可直接核对的产品规格书，来确认“Starlink 已公开采用硅基太阳能电池”。
- 因此，本 Task 对该命题的审计结论是：**证据不足，只能保留为低证据等级行业说法，不能升格为主文档事实口径。**
